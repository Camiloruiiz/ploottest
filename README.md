# PlootTest

PlootTest is the base platform for a simple e-commerce MVP documented in `docs/`.

## Local setup

1. Install dependencies with `npm install`.
2. Copy `.env.example` to `.env.local` and fill in the Supabase credentials.
3. Apply the schema in `supabase/schema.sql`.
4. Seed products with `npm run seed:products`.
5. Start the app with `npm run dev`.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run typecheck`
- `npm run test`
- `npm run test:e2e`
- `npm run test:ui`
- `npm run seed:products`
